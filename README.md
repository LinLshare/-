## 目的

放一些微信公众号开发中写的一些代码

## 已经实现的功能

- 自定义菜单的增加，删除和获取；
- 被动回复（文本、语音、单图文和多图文）；
- 天气预报功能（输入“广州天气”回复一条单图文，显示广州的天气概况）
- 增加笑话功能（输入“笑话”可以看笑话啦）
